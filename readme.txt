Allows administrators to impersonate any user securely from the Users list.

== Description ==

Impersonate is a clean, secure WordPress plugin that gives administrators the ability to log in as any other user. Ideal for support teams, developers, and site managers, this tool helps you troubleshoot user-specific issues without needing to reset passwords or ask for login access.

* One-click impersonation from the Users table
* Automatically restore admin session on logout
* Uses WordPress nonces for secure switching
* Requires 'manage_options' capability to use
* Works with PHP 8.3 and latest WordPress versions

== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Users' > 'All Users' and click 'Impersonate' next to a user.
